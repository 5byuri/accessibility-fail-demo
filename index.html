Pa11y WCAG violation demo site

A static, intentionally inaccessible test fixture for Pa11y, Pa11y CI, axe, and HTML CodeSniffer demos.

Training fixture only. These pages intentionally contain accessibility failures. Do not copy these patterns into production.

Run it locally

From this folder, start a local server and scan a page:

npm install
npm run serve
npm run test:all

# or scan a single page
npx pa11y http://localhost:8080/pages/all-in-one.html --runner axe --runner htmlcs --standard WCAG2AAA --include-warnings --include-notices

What “every WCAG violation” means here

WCAG failures are not a finite list of HTML snippets. Many require judgment about meaning, timing, task flow, content accuracy, captions, reading order, and assistive technology behavior. This fixture is a broad violation zoo, not a legal conformance benchmark.
